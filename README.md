# 한화캐롯 Brand Discovery AI Platform

관심사 기반 브랜드 발견 전략 플랫폼

## 구조

- **채널 성과 분석** — 캐롯 자동차보험 디지털 채널 진단
- **숏폼의 영향력** — 자동차보험 카테고리 숏폼 데이터
- **콘텐츠 전략** — Interest-driven 90/10 콘텐츠 전략
- **브랜드자산 분석** — 4개 핵심 자산의 관심사 기회 발견

## 브랜드 자산 (USP)

| 자산 | 레벨 | 확인된 기회 |
|------|------|-----------|
| ⏱️ 퍼마일 월정산 | CORE | 8개 |
| 🎀 여성/자녀 특약 | NEW | 5개 |
| 🤖 캐롯 테크 | TECH | 3개 |
| 🔧 보험서비스 | SERVICE | 2개 |

## 기술 스택

- Next.js 14 + React 18
- Claude API (AI 맥락 매칭)
- ListeningMind 데이터 (하드코딩)
- Vercel 배포

## 데이터 소스

- ListeningMind MCP — keyword_info, path_finder, cluster_finder
- 모든 검색 볼륨/인구통계 데이터는 하드코딩

## 개발

```bash
npm install
npm run dev
```
